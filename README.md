# Sobre o projeto

Estou desenvolvendo este projeto como uma forma de aplicar meus conhecimentos adquiridos até hoje para criar algo funcional e real que pudesse me ajudar no dia a dia. O objetivo principal dessa aplicação é criar as funcionalidades:

- Criar e excluir conteúdos estudados
- Criar um cronograma de revisões automaticamente
- Poder visualizar a evolução
- Criar metas
