# Sobre o projeto

Estou desenvolvendo este projeto como uma forma de aplicar meus conhecimentos adquiridos até hoje para criar algo funcional e real que pudesse me ajudar no dia a dia. O objetivo principal dessa aplicação é criar as funcionalidades:

- Criar e excluir conteúdos estudados
- Criar um cronograma de revisões automaticamente
- Poder visualizar a evolução
- Criar metas

## Descrição das funcionalidades

### Adicionar matérias estudadas

Essa funcionalidade permite a criação de matérias estudadas no dia.

<img width="1907" height="907" alt="image" src="https://github.com/user-attachments/assets/d79a091f-affe-4af8-b51e-02e056827d56" />

### Calendário indicando as próximas revisões

Essa funcionadade permite visualizar quais serão as datas que os conteúdos serão revisados. O calendário segue o método de revisões espaçadas, e ao criar uma matéria na aba de matérias, o calendário automaticamente 
calcula as datas das proximas revisões desse conteúdo.

<img width="1905" height="912" alt="image" src="https://github.com/user-attachments/assets/1826db03-e54a-41d6-8a25-912289f90f7d" />

### Avisos de revisões do dia

Quando existirem conteúdos a serem revisados no dia de hoje, o ícone do sino irá alertar se houverem revisões para o dia atual e também mostrará todos os conteúdos que deverão ser revisados neste dia.
